# Mochi
Mochi is a macOS menu-bar overlay pet.
It renders a pixel-art companion above the menu bar.
It reacts to CPU, RAM, and network activity.
Built with SwiftUI UI and an AppKit overlay window bridge.
Monitoring stays lightweight and uses public Apple APIs.
No data leaves your Mac; metrics stay local.
See docs/ARCHITECTURE.md for the planned modules.
See docs/DEVLOG.md for progress updates.
